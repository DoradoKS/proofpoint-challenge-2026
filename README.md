# Proofpoint Technical Challenge - 2026

This repository contains the complete solution for the Proofpoint technical assessment. The project is structured to demonstrate data processing capabilities, code quality, and technical communication.

## Repository Structure

The project is organized into the following components:

- **`PART_A.md`**: Technical and behavioral questions (English responses).
- **`exercise_1_streaming/`**: Part B - Data pipeline for cleaning and deduplicating a media catalog.
- **`exercise_2_words/`**: Part C - Word frequency analysis tool.
- **`.gitignore`**: Configuration to exclude temporary and local environment files.
